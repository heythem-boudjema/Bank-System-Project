This is the **advanced version** of the Bank Management System.  
It builds on the basic version by adding authentication, user roles, and permission-based management.

---

## Features

- All features from the Basic Version
- User login system
- Admin and user roles
- Permission-based access control
- User management (add/delete users)
- Enhanced transaction handling
- Console-based menu interface with role-specific access

---

## How to Run

1. Open the `v2-advanced-version` folder in your C++ IDE (Visual Studio, Code::Blocks, etc.)
2. Compile `main.cpp`
3. Run the program
4. Login as a user or admin, then follow the console menu instructions
5. Admins can add/delete users and access all functions; users have restricted access

---

- File handling for client and user data
- Console application

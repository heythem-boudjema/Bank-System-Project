# Bank Management System (C++)

A console-based bank management system written in C++.  
This repository contains three versions of the project, demonstrating incremental development and user interaction.

## Project Versions

- **01-basic-version/** – Core banking operations: client management, deposit and withdraw transactions.  
- **02-advanced-version/** – Adds user login, admin/user roles, and permission-based management.  
- **03-user-perspective/** – ATM simulation from the user's perspective with login and banking operations.

## How to Run

1. Open the desired version folder (`basic`, `advanced`, or `user-perspective`).  
2. Compile and run `main.cpp` using any C++ compiler.  
3. Follow the console menu instructions.
